## 🔧 Quick Commands

```bash
# One-time setup
cd backend
npm install
cp .env.example .env              # Edit with MongoDB URI
npm run seed

# Development
npm run dev                        # Terminal 1: Backend
cd ../frontend
python3 -m http.server 3000      # Terminal 2: Frontend

# Testing
curl http://localhost:3000/api/projects

# Or use Makefile shortcuts
make dev                          # Backend
make frontend                     # Frontend
make seed                         # Seed database
make test                         # Test API
```
